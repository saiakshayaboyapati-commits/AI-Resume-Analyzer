# AI Resume Analyzer

## Overview

AI Resume Analyzer is a Python-based web application that helps users analyze resumes against job descriptions using NLP techniques.

## Features

* ATS Resume Score
* Resume-JD Similarity Score
* Skill Gap Analysis
* AI Feedback
* Experience Indicator
* Skills Visualization (Pie Chart)
* PDF Report Download

## Technologies Used

* Python
* Streamlit
* PyPDF2
* Scikit-Learn
* Matplotlib
* ReportLab

## How It Works

1. Upload a PDF resume.
2. Paste a job description.
3. Analyze the resume.
4. View ATS score, similarity score, matching skills, and missing skills.
5. Download the analysis report as PDF.

## Installation

```bash
pip install streamlit PyPDF2 matplotlib scikit-learn reportlab
```

## Run the Application

```bash
streamlit run app.py
```

## Project Outcomes

* Implemented NLP-based resume analysis.
* Applied TF-IDF and Cosine Similarity.
* Built an interactive web application using Streamlit.
* Generated downloadable PDF reports.
